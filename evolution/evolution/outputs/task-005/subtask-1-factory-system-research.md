## 二、主流工厂管理系统分类

### 2.1 ERP 系统（企业资源计划）

| 系统名称 | 厂商 | 适用规模 | API 支持 | 备注 |
|----------|------|----------|----------|------|
| Kingdee K/3 Cloud | 金蝶 | 中大型企业 | REST API | 中国市场占有率约 13% |
| Yonyou NC/U8 | 用友 | 大型/集团企业 | REST/SOAP | 中国市场占有率约 30% |
| SAP Business One | SAP | 中小企业 | REST/DI API | 国际品牌，价格较高 |
| Odoo | Odoo SA | 中小企业 | XML-RPC/JSON-RPC | 开源，可自部署 |
| 鼎捷 ERP | 鼎捷软件 | 制造业专精 | REST API | 台湾品牌，制造业深耕 |

### 2.2 MES 系统（制造执行系统）

| 系统名称 | 厂商 | 核心功能 | API 支持 | 备注 |
|----------|------|----------|----------|------|
| Infor MES | Infor | 生产调度、质量追溯 | REST API | IDC MarketScape 领导者 |
| iFactoryApp | iFactory | 实时控制、AI 分析 | REST API | 支持 SAP/ERP 集成 |
| Critical Manufacturing | Critical | 工业 4.0、IoT 集成 | REST/GraphQL | 高端制造业 |
| Tulip Interfaces | Tulip | 无代码应用构建 | REST API | 适合中小厂商 |

### 2.3 替代方案（无数字化系统场景）

| 方案 | 描述 | 接入成本 | 适用场景 |
|------|------|----------|----------|
| 微信/钉钉机器人 | 通过企业微信/钉钉 API 接收生产报表 | 低 | 小作坊、手工记录 |
| 表单自动化（金数据/简道云） | 工厂填写在线表单，自动同步 | 低 | 半数字化过渡 |
| Excel/CSV 定时导入 | 约定格式，定时读取共享文件 | 极低 | 临时方案 |
| OCR 识别 + 人工确认 | 拍照识别生产单据 | 中 | 纸质单据场景 |

## 三、物流追踪 API 调研

### 3.1 国际物流追踪

| API 名称 | 覆盖承运商 | 定价模式 | 特点 |
|----------|------------|----------|------|
| 17TRACK | 3200+ | 免费试用 + 付费套餐 | 全球覆盖，支持批量查询 |
| AfterShip | 1000+ | 免费 tier + 订阅制 | 99.99% SLA，数据标准化 |
| Track123 | 数千 | 按查询量计费 | 支持 USPS/FedEx/UPS/DHL |

### 3.2 国内物流追踪（中国）

| API 名称 | 覆盖承运商 | 定价模式 | 特点 |
|----------|------------|----------|------|
| 快递 100 (Kuaidi100) | 2000+ | 免费额度 + 按量付费 | 国内最全，支持电子面单 |
| 快递鸟 | 1500+ | 免费 + 增值 | 支持在途监控、时效预测 |

### 3.3 快递 100 API 核心接口

```
基础查询：https://poll.kuaidi100.com/poll/query.do
订阅推送：https://poll.kuaidi100.com/poll/v2/register
电子面单：https://api.kuaidi100.com/ecom/order/new.do
```

**认证方式:** API Key + Customer ID  
**返回格式:** JSON  
**SDK 支持:** PHP, .NET, Python, Java

## 四、推荐方案

### 4.1 首选方案（需 Wilson 确认）

**假设:** 工厂使用金蝶/用友等主流 ERP

| 集成点 | 方案 | 数据字段 | 接入成本估算 |
|--------|------|----------|--------------|
| 订单状态 | ERP API 读取 | 订单号、产品、数量、计划交期 | 1-3 人天 |
| 生产进度 | MES API / ERP 工单模块 | 工单号、完成数量、合格率 | 2-5 人天 |
| 物流信息 | 快递 100 API | 运单号、承运商、轨迹、签收状态 | 0.5-1 人天 |

### 4.2 备选方案（无数字化系统）

| 集成点 | 方案 | 实现方式 | 接入成本估算 |
|--------|------|----------|--------------|
| 生产进度 | 企业微信/钉钉机器人 | 工厂每日填报，机器人推送 | 0.5-1 人天 |
| 物流信息 | 快递 100 API | 同上 | 0.5-1 人天 |
